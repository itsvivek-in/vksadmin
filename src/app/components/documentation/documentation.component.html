VKSAdmin

An Admin Panel Purely Made using Materialize CSS, for Angular Framework

About Angular:
  1. Angular v5
  2. Angular-Cli v1.7.3

Dependencies:
  1. jQuery v2.2.4 or Above
  2. Hammer JS v2.0.8
  3. Materialize v0.100.2
  4. Angular2-materialize v15.1.10

Plugins: (Already Installed With this Package)
  1. Chart JS v2.7.2
  2. Tinymce

How to Use: This project is only for the Angular users and is based on Angular 5
  1. Download this repository and extract the folder.
  2. Rename the folder, if you wish.
  3. Navigate to the folder using Terminal/Command Line and Run "npm serve" to start the project in developmnent mode.
